# QR Cafe App

This is a [Next.js](https://nextjs.org) project bootstrapped with React, Tailwind CSS, and using `.jsx` files.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.jsx`. The page auto-updates as you edit the file.

## Technologies Used

- **Next.js**: Framework for React applications.
- **React**: Library for building user interfaces.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Lucide React**: Icon library for React.
